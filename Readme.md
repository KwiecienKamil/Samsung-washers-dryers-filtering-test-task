## Instalation

1. Clone repository:
```
git clone https://github.com/KwiecienKamil/Samsung-washers-dryers-filtering-test-task.git
```
2. Install dependencies:
# Server
``` 
cd server
``` 
``` 
npm install
```  
``` 
npm start
``` 
# Client
``` 
cd ..
``` 
``` 
cd client
``` 
``` 
npm install
```  
``` 
npm run dev
```
